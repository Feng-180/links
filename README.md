# 资源风 (Resource Wind) - v12 🌸 樱花版

> 真正收录 300+ 全球优质极客资源的导航站，对标 FMHY 的中文版本。

## 🌸 v12 樱花版

全新 **樱花粉紫** 视觉风格，从蓝色科技风全面升级为唯美粉紫系配色。

- 🎨 **樱花粉紫配色** — 柔和粉色 + 高贵紫色渐变
- ✨ **极光背景** — 动态粉紫光晕效果
- 🪟 **毛玻璃卡片** — Glassmorphism 视觉设计
- 🚀 **零 Tailwind 依赖** — 纯 Vanilla CSS，加载更快
- 🔤 **高级排版** — Inter + Noto Sans SC 字体

## 🌟 项目特色

- **300+ 资源链接**：涵盖影视、动漫、游戏、音乐、软件、AI、开发工具等 13 个分类
- **现代化设计**：樱花粉紫主题、响应式布局、流畅动画
- **高效搜索**：实时搜索、分类筛选、随机访问
- **用户友好**：键盘快捷键 (Ctrl+K)、链接复制、移动端适配
- **轻量级**：无后端依赖，纯静态网站

## 📁 项目结构

```
links/
├── index.html              # 主页面（v12 樱花版）
├── index-optimized.html    # 同上（兼容旧链接）
├── data.json               # 资源数据库（JSON 格式）
├── CNAME                   # 自定义域名配置
├── README.md               # 本文档
├── bj.jpg                  # 背景图片
└── ndmz.jpg                # 装饰图片
```
