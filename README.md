# 资源风 (Resource Wind) - v11 优化版

> 真正收录 300+ 全球优质极客资源的导航站，对标 FMHY 的中文版本。

## 🌟 项目特色

- **300+ 资源链接**：涵盖影视、动漫、游戏、音乐、软件、AI、开发工具等 13 个分类
- **现代化设计**：深色主题、响应式布局、流畅动画
- **高效搜索**：实时搜索、分类筛选、随机访问
- **用户友好**：键盘快捷键、链接复制、移动端适配
- **轻量级**：无后端依赖，纯静态网站

## 🔧 开发指南

### 添加新资源

编辑 `data.json`，按照以下格式添加：

```json
{
  "c": "movies",           // 分类代码
  "t": "资源名称",         // 资源标题
  "d": "资源描述",         // 资源描述
  "u": "https://...",      // 资源链接
  "i": "fa-film",          // Font Awesome 图标
  "badge": "热门"          // 可选：标签
}
```

### 分类代码
- `movies` - 影视与聚合
- `anime` - 动漫与漫画
- `games` - 游戏与模拟
- `music` - 音乐与有声
- `iptv` - 直播与IPTV
- `tools` - 软件与工具
- `ai` - 人工智能
- `dev` - 开发与编程
- `design` - 设计与素材
- `android` - 安卓发烧友
- `privacy` - 隐私与安全
- `pan` - 网盘与资源

### 可用的 Badge 类型
- `热门`、`神器`、`必藏`、`神站` - 红色
- `新锐`、`FOSS`、`推荐`、`源头`、`必读` - 绿色
- 其他 - 蓝色
